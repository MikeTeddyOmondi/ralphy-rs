# Example Project - Task Management App

A simple task management application built with Rust and web technologies.

## Project Goals

Build a full-stack task management application that demonstrates modern development practices.

## Tasks

### Backend

- [ ] Set up Rust project with Actix-web
- [ ] Create PostgreSQL database schema
- [ ] Implement user authentication with JWT
- [ ] Add CRUD endpoints for tasks
- [ ] Add task filtering and search
- [ ] Implement task priorities and tags
- [ ] Add API rate limiting
- [ ] Write integration tests

### Frontend

- [ ] Set up React project with TypeScript
- [ ] Create login/signup pages
- [ ] Build task list component
- [ ] Add task creation form
- [ ] Implement task editing and deletion
- [ ] Add drag-and-drop task reordering
- [ ] Create task filtering UI
- [ ] Add dark mode support
- [ ] Write unit tests with Jest

### DevOps

- [ ] Create Dockerfile for backend
- [ ] Create Dockerfile for frontend
- [ ] Set up docker-compose for local development
- [ ] Configure GitHub Actions CI/CD
- [ ] Add code coverage reporting
- [ ] Set up staging environment
- [ ] Configure production deployment

### Documentation

- [ ] Write API documentation
- [ ] Create setup guide
- [ ] Add architecture diagrams
- [ ] Write deployment guide

## Technical Stack

- **Backend**: Rust, Actix-web, PostgreSQL
- **Frontend**: React, TypeScript, TailwindCSS
- **DevOps**: Docker, GitHub Actions
- **Testing**: Jest, cargo-test

## Success Criteria

- All endpoints return proper status codes
- Test coverage > 80%
- Page load time < 2 seconds
- Mobile responsive design
- Accessible (WCAG 2.1 AA)
